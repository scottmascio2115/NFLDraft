NFL Mock Draft

How to Begin the Draft:
- Clone the repository
- Bundle Install
- rake db:create
- rake db:migrate
- Import Players, Teams and Order by running rake load:all
- Run Rails s and go to http://localhost:3000/ to See the Order of the Draft and the Available Players
- Begin Draft rake draft:start
- Output of draft is displayed in the console and in the browser


Run the Test Suite
- rake db:test:prepare
- rake
